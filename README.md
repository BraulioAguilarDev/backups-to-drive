# Upload file to Google Drive

Go tool to upload files

## Usage

```shell
$ go run *.go -i ${file} -o ${description}  -f ${folder}
```

| Flag | Required | Description          |
|------|----------|----------------------|
| -i   | True     |     My file          |
| -o   | False    |     Description file |
| -f   | False    |     Folder           |
